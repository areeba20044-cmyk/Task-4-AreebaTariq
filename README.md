# Bahria Living

## Project Title
Bahria Living – Premium Real Estate Pakistan

## Description
A polished real estate landing page built for Bahria Town properties across Pakistan. The page showcases featured property cards, a branded hero section, interactive navigation, and luxury design elements for a high-end property presentation.

## Built With
- HTML5
- CSS3
- JavaScript

## Important Features
- Responsive landing page layout
- Animated page loader
- Smooth scrolling navigation
- Featured property cards with badges
- Hero section with premium branding and action buttons
- Custom scrollbar styling
- Interactive UI details and hover effects

## Important Details
- No external build tools are required; this is a static front-end project.
- All assets are built with standard web technologies and hosted locally.
- The current code base includes:
  - `index.html` — page structure and content
  - `style.css` — styling, layout, and visual theme
  - `script.js` — page interactions, loader, nav, and effects

## How It Is Built
1. `index.html` defines the page sections, navigation, hero, property cards, and page structure.
2. `style.css` applies the dark luxury theme, typography, responsive layout, and visual polish.
3. `script.js` handles the loader animation, smooth anchors, navigation behavior, and UI interactions.

## How to Run Locally
1. Open `index.html` in a web browser.
2. No server is required for local testing, but you can use any static server if preferred.
